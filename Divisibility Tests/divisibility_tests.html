Divisibility Tests 2-1000

Divisibility Rule for 11:
If the difference between the sum of the digits in odd positions and the sum of the digits in the even positions is either 0 or divisible by 11, then the number is divisible by 11.

247588 => 2+7+8=17 => 4+5+8=17 => 17-17=0

∴ 247588/11=22508

227480 => 2+7+8=17 => 2+4+0=6 => 17-6=11

∴ 227480/11=20680

Divisibility Rule for 12:
If the number is divisible by 3 and 4, then the number is divisible by 12.

4572 => 4+5+7+2=18 => 18/3=6

4572 => last two digits 72 => 72/4=18

∴ 4572/12=381

Divisibility Rule for 4:

Divisibility Rule for 5:

Divisibility Rule for 6:
If the number is divisible by both 2 and 3, then the number is divisible by 6.

576 => last digit is even (6) => 576/2=288

576 => 5+7+6=18 => 18/3=6 => 576/3=192

∴ 576/6=96

Divisibility Rule for 7:
Take the last digit, double it, and subtract it from the rest of the number. If that answer is divisible by 7, then the original number is divisible by 7.

24003 => last digit is 3 => 3*2=6 => 2400-6=2394 => 2394/7=342

∴ 24003/7=3429

Divisibility Rules for 13:
There are four methods to test for divisibility by 13.

Method One: Alternating Sum of Triplets (especially useful for large numbers)

5,098,418 => 418, 098, 5 => 418-098+5=325 => 325/13=25

∴ 5,098,418/13=392,186

Method Two: Multiply the Last Digit by 4

6760 => Last digit is 0 => 676+(0*4)=676 => 676/13=52

∴ 6760/13=520

Method Three: Subtracting the Last Two Digits and Subtract From Four Times the Rest of the Number

17537 => Last two digits are 37; remaining number is 175 => 175*4-37=663 => 663/13=51

∴ 17537/13=1349

Method Four: Multiply the Last Digit by 9 and Subtract From the Rest of the Number

884 => 88-(4*9) => 88-36=52 => 52/13=4

∴ 884/13=68

Divisibility Rule for 14:
There are two rules for testing divisibility by 14.

Method One: Test for Divisibility by Both 2 and 7

7546 => 7546 is an even number => 7546/2=3773

7546 => last digit is 6 => 6*2=12 => 754-12=742 => 742/7=106

∴ 7546/14=539

Method Two: Add the last two digits of the number to twice the rest of the digits.

19,460 => 194 60 => (194*2)+60 => 388+60=448 => 448/14=32

∴ 19460/14=1390
